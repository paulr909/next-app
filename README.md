# Next.js App (v13)

Install dependencies:
```bash
npm install
# or
yarn
```

Run the development server:
```bash
npm run dev
# or
yarn dev
```

Build the project:
```bash
npm run build
# or
yarn build
```

Run production build:
```bash
npm run start
# or
yarn start
```
